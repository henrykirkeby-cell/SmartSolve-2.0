How to install the extension

Takes about 2 minutes — no tech skills needed
1

Download the extension

Click the button below to download the SolveMate extension files to your computer.

⬇ Download SolveMate Extension
File is a .zip — you'll extract it in the next step
2

Extract the zip file

Find the downloaded file (usually in your Downloads folder), right-click it, and select Extract All. Remember where you put the extracted folder.

3

Open Chrome Extensions

In Chrome, go to chrome://extensions in the address bar, or click the three dots menu → More tools → Extensions.

4

Enable Developer Mode

In the top right corner of the Extensions page, toggle on Developer mode. New buttons will appear at the top left.

5

Load the extension

Click Load unpacked, then select the extracted folder from Step 2. SolveMate will appear in your extensions list!

6

Pin it and log in

Click the puzzle piece icon 🧩 in Chrome's toolbar, find SolveMate, and click the pin icon. Then click the SolveMate icon and log in with your account.
